# Vigenere Autokey System (Turkish Support)

Bu proje, C++ ve Win32 API kullanılarak geliştirilmiş bir şifreleme aracıdır.

## Özellikler
- **Autokey Algoritması:** Mesajın kendisini anahtar olarak kullanarak güvenliği artırır.
- **Türkçe Karakter Desteği:** Alfabe 29 harf üzerinden (mod 29) hesaplanır.
- **GUI:** Kullanıcı dostu grafiksel arayüz.

## Derleme (Compilation)
`g++ lab1.cpp -o vigenere -lgdi32 -luser32`
